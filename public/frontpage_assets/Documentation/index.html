Installation

To install NextCoin on your web server, download the files from Themehunt and unzip them to the local disk on your computer. Inside the extracted folder you will see three folders named NextCoin - Documentation. NextCoin - Template and Custom Line Icons.

NextCoin - Template folder structure is as follows:

  • css - contains all verndor and custom css files
  • fonts - contains icon fonts
  • img - contains all images used in the template
  • js - contains all vendor and custom js files
  • index.html - Home Page

Using an FTP Client (such as Filezilla or Total Commander), you will need to upload all of the contents of the NextCoin - Template folder to your server. And other files can be used according to your preferences.

HTML Structure

This template is based on a responsive and full screen layout. And it's built using HTML5 standards.

index.html

From beginning of the <body> tag till the ending </body> tag, all the codes and texts are used for the template structure. You can change any text from the template by navigating through the index.html file and localizing each line of text you want to change.

Like index.html, you can change any page included with this template starting from <body> tag till ending </body> tag.

Change Placeholder Images

When you download the template files you will see we used placholder images instead of those originals we used in our preview link. To change with your desired images replace the placehodlder images. If you face any problem with where you have to use images, please see our live preview to get idea Demo

Dark Style

If you want to use the dark style of this template, please rename style.css to style-dark.css in line number 20 of the template files.

Logo

By default, this template has text based logo. If you wish to use an image based logo, simply delete NextCoin text from line number 44, uncomment line number 46 and place your logo into img folder.

Contact form

This template comes with a working contact form. For this we need php support - so if you try this on your personal computer it will not work (unless you have Apache server or something like that running). You have to try it on some web server and this server must have php support.

We've included a contact form PHP file which is email.php in the root of this template folder. You have to simply change your Email Address in the file to start receiving Form Responses. Replace email@domain.com with your email id in line number 6.

Please DO NOT play aroundwith this file unless you have PHP coding experience.

CSS Files

All the css files are located in the css folder of the template.
style.css - Main stylesheet which includes all the styles of the template.
style-dark.css - Stylesheet for dark style template.
custom.css - Your own custom stylesheet. If you would like to edit a specific section of the template, simply find the appropriate labels and add respective styles there.

JavaScript Files

All vendor, 3rd party plugin and our custom JavaScript files are located in the js folder.
plugins.js - Includes all 3rd party plugins in the template.
main.js - Our custom JavaScript file. You can add your custom js codes there if you need to.

Please DO NOT edit this files unless you have JavaScript coding experience. Simple mistakes can cause serious damage to the scripts
